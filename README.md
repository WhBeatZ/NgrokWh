# NgrokWh

El Mejor Ngrok para termux, 100% Funcional. Creado por el famoso Sensei, esta herramienta se usará también en WhPhisher

# Como instalar


`pkg update && pkg upgrade`

`ppkg install bash`

`pkg install git`

`git clone https://github.com/WhBeatZ/NgrokWh`

`cd NgrokWh`

`chmod +x ngrokwh.sh`

`./ngrokwh.sh`

Y Disfrutar :D
